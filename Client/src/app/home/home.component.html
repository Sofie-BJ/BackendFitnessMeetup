Welcome to the home of Fitness Meetup

This web site uses the following technologies:

  • .NET Core for the server
  • Angular 6 for the front end
  • Entity Framework Core for the repository

This project is based on the .Net Core Angular template

Supported features

  • Facebook SSO
  • Posting new meetups
  • Joining a meetup
  • Viewing upcoming meetups and sorting them by sports type
